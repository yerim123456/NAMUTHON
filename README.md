# NAMUTHON
2023_NAMUTHON_왕꿈트리

## 문제
- 2030 청년층을 대상으로 한 전세 사기 증가
- 자금력이 부족하고 부동산 거래 등을 포함한 사회 경험이 적은 청년층은 사기성 거래에 취약할 수밖에 없음

## 기획 의도
1. 사전 예방을 통한 안전한 거래 환경 조성
2. 금융 교육/자금 지원 관련 콘텐츠 제공을 통한 정보 비대칭 해소  
3. 2030 청년층을 위한 맞춤형 정보 제공
